1. Дан код:

        var a = 1, b = 1, c, d;
        c = ++a; alert(c); // 2
        d = b++; alert(d); // 1
        c = (2+ ++a); alert(c); // 5
        d = (2+ b++); alert(d); // 4
        alert(a);
        alert(b); 


Сперва переменным а и b присваивается значение 1, а переменные c и d
просто объявляются, затем переменной с присваивается значение инкрементированной
переменной a, с помощью alert выводится полученное значение 2,
переменной d присваивается значение переменной b, равной 1, которая в дальнейшем
инкрементируется, alert(d) дает 1, переменной c присваивается значение выражения 
(2 + 3), т.к. а, равная двум, инкрементируется, alert(c) дает 5-ку, переменной d
присваивается значение выражения (2 + 2), т.к. b была инкрементирована во втором
примере и равна 2, alert(d) дает 4, alert(a) дает 3, т.к. после третьего примера
больше не изменялась, а alert(b) дает 3 после постфиксного инкремента в 4-м примере
